# 世界游戏

一个类似minecraft的极简沙盒模拟游戏，用于测试。

## 世界规则
离散世界，离散移动，离散时间。
每tick执行世界运算，规则书里所有a-b都代表a<=x<=b的均匀整数分布。
每t，周围四格内无任何其他方块、且不与玩家坐标相同的树苗方块有10%概率变成树，就是垂直2-4格木板+1格树叶的结构。

## 物品

树苗：只能作为高度0的方块放置，物品堆叠8
木板：可放，物品堆叠8
木斧：不可放，物品堆叠1。

## 方块

| 物品名字 | 破坏时间/t | 掉落物 | 掉落数量 | 有无实体 |
|--------|----------|------|--------|--------|
| 树苗 | 1 | 树苗 | 1 | 无 |
| 木板 | 4 | 木板 | 1 | 是 |
| 树叶 | 1 | 树苗 | 1-3 | 是 |

## 合成

木斧：需3木板 消耗2t

## 玩家
0. 玩家高度为2，高度为2的空间内有障碍时玩家无法跨越。
1. 玩家移动速度 1格/t
2. 玩家物品栏有10格（0-9），其中第0格为主手，每格最多可以放一堆叠同类的物品，不同的物品必须放在不同格中。
3. 玩家可以任意交换物品栏物品的位置，不消耗t。
4. 玩家主手是木斧时，挖掘木板速度翻倍；玩家主手是物品时，才可能可以放置物品。

玩家只能破坏自己面前与自己同高或高度+1、或自己脚下的方块（玩家破坏自己脚下的方块时，玩家高度-1）。
玩家只能同时破坏一个方块，否则会重置破坏并重新计算破坏时间。
玩家只能放置自己面前与自己同高或高度+1、或自己脚下的方块（在自己脚下放置有实体方块时，玩家高度+1，相当于垫起，前提是玩家头顶必须没有方块遮挡才能垫起；放无实体方块时，玩家高度不变），并且必须与已经存在的方块相邻，或高度=0的第一层上放置，不能悬空放置。
玩家放置任何方块都需要消耗1t。

5. 玩家具有朝向，朝向分为东南西北四个方向，转到任何方向都要1t，玩家只能在朝向的前方放置或破坏方块，玩家只能向朝向的方向移动.

## 初始

坐标格式为(x,y,z)，其中x轴正方向为东，y轴正方向为南，z轴正方向为上。
世界大小5\*5\*5，最中心点的坐标是0,0,2。
玩家初始位置为(0,0,0)，朝向东，物品栏内只有一个树苗。
