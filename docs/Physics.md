# Physics

物理刻的特点：它在玩家引起的MutationGroupSequence与世界引起的MutationGroupSequence形成后，GameTick已经执行了具体一个MutationSequence，世界已经发生了确定的改变之后开始执行。do_physics是World的方法。

物理刻的特点是确定性与合理性：确定性指的是物理刻的执行结果是确定的，不会再产生新的可能性；合理性指的是，物理刻的执行目的是为了消除上面的所有Mutation可能产生的冲突（比如玩家刚好走进了一个刚长成的树的方块上，或玩家挖掉了自己脚下的方块高度减了1），让世界合理运行。

目前物理刻引入了三种概念：重力、碰撞、边界挤压。
重力：当玩家脚下没有方块时，玩家自由落到脚下最高的方块的高度。
碰撞：当在某种情况下玩家与一个实体方块重合时（比如玩家恰好走到了突然长成的树方块中），玩家会被方块挤开到四周最近的一个可以容纳玩家的位置。具体的算法是，从玩家当前位置开始 BFS，按上左右下的顺序推出，直到can fit player。
边界挤压：当玩家走出了世界边界，则他需要回到离他最近的世界内格子的位置。

世界PhysicsTick的运行过程：只要世界还不是合法状态，就反复执行重力+碰撞+边界挤压，直到世界合法为止。如果超过一定次数还没有合法，则抛出异常。
